# Burger Blast – Mobile Menu UI Design

Burger Blast is a mobile-first UI design and static front-end implementation for a burger restaurant ordering application.
This project was completed as part of a UI/UX and Web Development coursework exercise, with a strong focus on layout design, visual hierarchy, and responsive styling using HTML and CSS only.

# Important:
This design is original and not copied from any existing burger or food delivery applications, in accordance with course academic integrity requirements.

# Project Objectives
Design a complete mobile restaurant menu interface
Create visually finished screens for:
       Menu browsing
       Item description
       Shopping cart
Implement designs as static HTML pages
Use CSS only for layout, styling, and responsiveness
Simulate user interactions visually (no JavaScript logic)

# App Features & Screens
# Menu Page
Category navigation (Burgers, Sides, Beverages, Desserts)
Menu item cards with:
         Name
         Rating
         Price
        Image placeholder
        “Add to cart” button (visual only)
Mobile-friendly grid layout
Search bar and filter icon (visual only)

# Item Description Page
Large product image
Product name header
Quantity selector (non-functional visual control)
Detailed sections:
      Concept description
      Ingredients list
      Estimated nutritional values (simplified FDA-style format)

# Shopping Cart Page
List of selected items with:
        Image
        Item name
        Quantity
        Individual price
Restaurant location selector
Total cart amount
Primary checkout button (visual only)

# Technologies Used
HTML5
CSS3
Flexbox & Grid layout
Media queries for responsive design
Bootstrap 5 (for basic layout utilities)
Google Fonts
Figma (UI design & prototyping)
GitHub Pages (public hosting)

# Design Approach
Mobile-first UI design
Clean, minimal layout to keep focus on food items
Consistent color palette and branding
Clear visual grouping of elements
Reusable CSS classes for maintainability
Visual feedback through hover and active states (CSS only)
